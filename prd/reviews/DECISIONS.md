# 決策紀錄（DECISIONS）

> 本檔由 Synthesizer 維護，用於記錄「採納/拒絕/延後」的原因，讓 PRD 變更可追溯。

## 決策格式（建議）

每條決策請包含：
- Decision ID（例如 D001）
- 日期
- 上下文（引用 PRD §X.Y 或 review 檔的段落）
- 選項（A/B/折衷）
- 決策（採納/拒絕/延後）
- 理由（含取捨）
- 回退策略（如果採納後出問題怎麼回退）
